# CANON_AUTHORITY.md

When campaign files disagree, use this authority order:

1. User's direct instruction
2. CLAUDE.md
3. /00_control/CANON_AUTHORITY.md
4. /03_canon/CANON.md
5. /03_canon/DM_ONLY_CANON.md
6. /03_canon/PLAYER_SAFE_CANON.md
7. Runtime state files in /02_runtime_state
8. Current stage files
9. Older quest/location/NPC/faction files
10. Generated drafts, TODOs, and backlog notes

Do not resolve contradictions silently. If a contradiction affects canon, record it in CONSISTENCY_AUDIT.md and either fix it or ask for a decision if it is truly ambiguous.
