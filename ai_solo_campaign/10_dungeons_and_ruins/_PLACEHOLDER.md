# _PLACEHOLDER.md — /10_dungeons_and_ruins/

## Purpose

This folder will contain files for dungeons, ruins, adventure sites, and other explorable locations beyond standard settlements.

## Current Status

**Empty — Stage 0 scaffold.** Dungeon/ruin files will be created during Stage 3 (5–8 dungeons) and Stage 12 (full adventure site expansion).

## Expected Contents

Each dungeon or ruin file should include:

- Purpose in the world
- History
- Entrance and approaches
- Occupants
- Zone or room list
- Hazards
- Secrets and clues
- Puzzles
- Treasure
- Boss or climax encounter
- Noncombat options
- Solo-play safety valves
- Consequences if ignored
- State update triggers

## File Naming Convention

`[location-name].md` — e.g., `the-bleached-tower.md`, `sunken-hall-of-orrath.md`
