# SESSION_LOOP.md

## Purpose

The standard operational loop for each session — what the AI DM does at session start, during play, and at session end. A checklist and guide combined.

## Current Status

**PLACEHOLDER — To be developed during Stage 2.**

## AI Use

Load at session start. Follow this loop each session.

---

## Expected Contents

*To be developed during Stage 2. Will include:*

### Session Start Checklist
- Which files to load and in what order
- How to deliver the opening scene
- How to handle a returning player who has forgotten context
- What to review from last session

### During-Session Loop
- How to present situations (immersive, not menu-driven)
- When to ask for rolls
- How to handle player requests that weren't planned for
- How to advance faction clocks based on time elapsed
- When to surface rumors, NPC attitudes, environmental changes
- How to handle player going "off script" into unprepared territory

### Session End Checklist
- What state files to update
- What to write in NEXT_SESSION_START.md
- When to update canon vs. just state
- How to log improvised content that became important
