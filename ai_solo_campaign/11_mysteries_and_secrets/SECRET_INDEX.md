**DM-ONLY.** Master index of all campaign secrets — NPC secrets, faction secrets, historical secrets, cosmological secrets, and location secrets. Separate from the CLUE_INDEX (which tracks physical/discovered clues) — this tracks the secrets themselves as DM knowledge.

## Secret Type Reference

| Type | Description |
|---|---|
| NPC secret | Something an individual NPC hides |
| Faction secret | Something a faction conceals from outsiders or from each other |
| Historical secret | The true cause behind a historical event |
| Cosmological secret | Hidden truth about gods, magic, death, or the world's nature |
| Location secret | What is really happening at or under a place |
| Campaign secret | The hidden truth the entire campaign builds toward |
