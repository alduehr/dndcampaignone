# _PLACEHOLDER.md — /07_factions/major_factions/

## Purpose

This folder will contain one file per major faction in the campaign world.

## Current Status

**Empty — Stage 0 scaffold.** 5–8 major factions will be created during Stage 1.

## Expected Contents

Each major faction file should follow the Faction File Standard defined in `00_control/FACTION_STANDARDS.md`:

- One-sentence identity
- Public face
- True agenda (DM-only)
- Founding / history
- Leader
- Important members
- Resources
- Methods
- Territory / influence
- Internal conflict
- Allies and enemies
- Secrets
- Current activity
- Escalation clock
- Quests offered and quests against
- How they react to the player
- What they do if ignored
- Solo-player safety considerations

## File Naming Convention

`[faction-name].md` — e.g., `the-iron-compact.md`, `order-of-the-pale-flame.md`
