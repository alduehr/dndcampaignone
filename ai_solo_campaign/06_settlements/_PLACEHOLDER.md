# _PLACEHOLDER.md — /06_settlements/

## Purpose

This folder will contain one file per named settlement — cities, towns, villages, and outposts.

## Current Status

**Empty — Stage 0 scaffold.** Settlement files will be created during Stage 1 (starting settlement) and Stage 3 (regional expansion).

## Expected Contents

Each settlement file should follow the Settlement Standard defined in `00_control/WORLDBUILDING_STANDARDS.md`:

- One-sentence identity
- First impression
- Public overview
- DM-only truth
- Districts / notable areas
- Local leadership
- Important NPCs
- Factions present
- Shops and services
- Laws and customs
- Current tensions
- Secrets
- Rumors
- Quest hooks
- Possible scenes
- Consequences if ignored

## File Naming Convention

`[settlement-name].md` — e.g., `thornhaven.md`, `the-city-of-dusk.md`

## Stage 1 Target

The starting settlement will be established during Stage 1. Additional settlements are developed in Stage 3.
